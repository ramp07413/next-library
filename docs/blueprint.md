# **App Name**: LibMan Platform

## Core Features:

- User Authentication: Secure login and registration system for company admins, library admins, and students.
- Dashboard Views: Customizable dashboards for company, library, and student roles, each displaying relevant KPIs and information.
- Library Management: CRUD operations for managing libraries, including details, users, and settings.
- Seat Management: Tools for creating and managing seats within libraries, including seat allocation and availability status.
- Payment Tracking: System for tracking student payments, dues, and transaction history.
- Alert & Notification System: Push alerts to users regarding various messages, like overdue payments, and new announcements. Uses an LLM tool to assess which notifications are most critical for each user.
- Reporting & Analytics: Generation of reports and analytics based on library data, including revenue trends, user activity, and seat occupancy. CSV/PDF export available

## Style Guidelines:

- Primary color: Deep blue (#2962FF) to convey trust and professionalism.
- Background color: Light gray (#F5F7FA) to provide a clean and modern look.
- Accent color: Teal (#30D5C8) to highlight key actions and elements.
- Body and headline font: 'Inter', a sans-serif font known for its legibility and clean design, will be used throughout the platform.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use simple and consistent icons from a library like FontAwesome or Material Icons to ensure clarity and ease of understanding.
- Implement a responsive layout with a sidebar navigation and top navbar to provide intuitive access to all features, even on mobile devices.
